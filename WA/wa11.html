Random Anime Quote
I extended this qoute machine in a two ways. First, when you click the new qoute button it changes the qoute, character, anime and it will also change the color of the box (.app). This was done through a random number generator that takes a number from the length of the colors array we made and we give to the function. Second, I added a change background button that takes from the same list as the box color changer and changes the background of the page when you click on it to a random color I have set in an array.
